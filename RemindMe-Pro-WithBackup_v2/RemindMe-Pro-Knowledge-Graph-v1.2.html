🔔 FIXED v1.2: Multi-Fire Notification Timing

scheduleMultiFire(task)  ← caller checks isNotificationsDisabled() first
       │
       ├── remindUntilDone === true ──→ UNTIL-DONE MODE
       │     Wait until dueDate → fire → repeat every interval until ack
       │
       └── remindUntilDone === false ──→ NORMAL MODE (FIXED)
             │
             ▼
           firstFireTime = dueDate - (count-1) × intervalMs
             │
             ▼  Example: Due 3:00 PM, 5 reminders × 15 min
                firstFireTime = 3:00 - (4 × 15min) = 2:00 PM
             │
             ▼
           for i = 0 → count:
             fireTime = firstFireTime + (i × intervalMs)
             delay = fireTime - now
               ├── delay < -60s → SKIP (past, don't spam)
               └── delay ≥ 0   → setTimeout → check ack → notify
             │
             ▼
           BEFORE FIX: 3:00→3:15→3:30→3:45→4:00 ← all AFTER ❌
           AFTER FIX:  2:00→2:15→2:30→2:45→3:00 ← last AT due ✅

⏳ NEW v1.1: Countdown Timer Flow

renderHome() → clear previous interval → render badges
       │
       ▼
setInterval(updateCountdowns, 1000)  ← every second
       │
       ▼
Each tick, for each [data-countdown-target] badge:
  diff = target - Date.now()
  ├── diff ≤ 0    → "⚠️ OVERDUE"    + .countdown-overdue
  ├── diff > 24h  → "⏳ 3d 5h 20m"  (calm purple)
  ├── diff > 1h   → "⏳ 5h 30m 15s" (calm purple)
  ├── diff ≤ 5min → "⏳ 4m 22s"     + .countdown-urgent (red pulse)
  └── diff < 1min → "⏳ 45s"        + .countdown-urgent (red pulse)

🔇 NEW v1.1: Disable Notifications Flow

Toggle ON in Settings
  → localStorage('remindme-notifications-disabled') = 'true'
  → cancelReminders() for ALL tasks
  → Grey out browser permission row

Effects across codebase:
  • init()           → skip requestPermission()
  • scheduleExisting → skip entirely
  • create/edit task → skip scheduleMultiFire()
  • reminder-fired   → ignored (no toasts)

Toggle OFF → "restart app to reschedule"

Task Creation Flow (with notif check)

FAB → dialog (CREATE) → fill → submit → validate
  → generateId() → addTask() → IndexedDB
  → isNotificationsDisabled()? YES: skip | NO: scheduleMultiFire()
  → dialog.close() → toast → navigateTo('home')
