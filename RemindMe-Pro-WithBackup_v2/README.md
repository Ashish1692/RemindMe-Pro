# RemindMe Pro 🔔

Fully functional reminder PWA with **Vanilla JavaScript** — zero dependencies.

## Features
- ✅ Create, complete, snooze, delete reminders
- 🔔 Multi-fire notifications (1-20x)
- 🔁 "Remind Until Done" mode
- 💾 **Import/Export backup (JSON & CSV)**
- 🔄 **Auto-backup every 30 minutes**
- 🌙 Dark/Light mode
- 📁 Categories | 📅 Calendar
- 📱 PWA — installable, works offline

## Quick Start
```bash
npx serve .
# or: python3 -m http.server 3000
```
Open http://localhost:3000
